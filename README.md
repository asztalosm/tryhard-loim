# specs
- 15db egyre nehezedő kérdés, konzolon keresztül kell feltenni (no gui :( )
- helyes válasz esetén következő kérdés, helytelen válasz esetén vége
- minden 5. helyes kör után fix nyeremény, utána választhat, hogy folytatja, v abbahagyja, ha 9-nél száll ki akkor az 5.-et kapja meg, ha 10-nel akkor 10.-et
- 4db válaszlehetőség, ABCD válasz
- 3 segítség: FKT felező közönség telefon, mindegyik csak egyszer jelenhet meg és egyszer lehet használni
